ScaleType:
     - fitcenter: 缩放图片到view得尺寸
     - fitstart: 靠上显示
     - center： 不会缩放，只会截取（只能显示出一部分）
     - center insight：将图片等比例缩小，直到他能显示在中间。如果本来就小就不变
     - centercrop： 缩放后将整个imageView填满
